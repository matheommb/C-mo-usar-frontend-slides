02 · Solución
La nueva forma

Contexto en archivos
+ un comando =
presentación lista.

# invoca la skill desde Claude Code
$ /frontend-slides Genera el informe mensual de cartera usando los archivos en /contexto

Claude lee los archivos, interpreta los datos y genera un HTML completo con navegación, animaciones y formato consistente — listo para abrir en el browser.

03 · Los 7 pasos

La skill ejecuta 7 pasos automáticos.

01
Detectar el modo
Nueva presentación · Conversión PPT · Mejora de slides
02
Descubrir el contenido
Propósito · Extensión · Estado del contenido
03
Descubrir el estilo
3 previews visuales · Selección de preset · Mezcla de estilos
04
Construir la presentación
HTML semántico · CSS variables · JS de navegación · Animaciones
05
Forzar viewport fit
100dvh · overflow:hidden · clamp() en todo · Split si no cabe
06
Validar en múltiples tamaños
Desktop · Tablet · Mobile · Landscape phone
07
Entregar
Limpia previews · Abre en browser · Resume path y customización
04 · Contexto

Estructura tu carpeta
de contexto así.

informe-cartera/
  contexto/
    resumen.md ← propósito e instrucciones
    datos.csv ← métricas clave
    hallazgos.md ← análisis del equipo
    audiencia.md ← para quién es el informe
  assets/ ← imágenes opcionales
  CLAUDE.md ← preset y reglas del equipo
resumen.md
Propósito del informe, período, decisiones que se tomarán con él y a quién va dirigido.
datos.csv / .xlsx
Cifras concretas. Claude las lee, interpreta tendencias y las convierte en slides con jerarquía visual.
CLAUDE.md
Fija el preset del equipo, el tono y las reglas de formato. Claude lo aplica sin necesidad de pedirlo cada vez.

06 · Resultado

Un archivo HTML.
Cero dependencias.
Todo adentro.

  • ▹
    Navegación completa — teclado (←→), rueda del mouse, swipe táctil, barra de progreso superior.
  • ▹
    Animaciones en entrada — cada elemento aparece con timing calculado usando IntersectionObserver, sin ser ruidoso.
  • ▹
    100% responsive — se ve igual en 4K, laptop, tablet y celular. Todo con clamp().
  • ▹
    Reducción de movimiento — respeta prefers-reduced-motion automáticamente.
  • ▹
    Abre en el browser sin servidor — doble clic al archivo. Sin instalar nada externo.
07 · Consistencia
El verdadero valor

Activa el estándar
del equipo.

Formato visual unificado
Cada persona elige colores y fuentes distintos.
Un preset en CLAUDE.md — mismo look en todos los informes del equipo.
Estructura de slides fija
Cada informe tiene un orden diferente de slides.
Siempre: título → problema → datos → hallazgos → próximos pasos.
Actualización instantánea
Un dato cambia → rehace el deck completo a mano.
Actualiza el .csv → ejecuta el comando → presentación rehecha en minutos.
Foco del equipo en el análisis
El equipo pierde tiempo en diseño y formato visual.
El equipo se enfoca en los datos. Claude decide cómo se ve.
