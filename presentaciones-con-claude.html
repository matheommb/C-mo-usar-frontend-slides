01 · Problema
El estado actual

Crear informes es
trabajo manual costoso.

⏱
Tiempo perdido
Horas copiando datos a slides, ajustando formato, alineando colores.
🎨
Inconsistencia visual
Cada persona hace la presentación a su manera. No hay un estándar.
🔁
Reproceso constante
Pequeños cambios de datos obligan a rehacer slides manualmente.
📂
Contexto disperso
La información vive en correos, Excels y chats — nunca centralizada.
02 · Solución
La nueva forma

Contexto en archivos
+ un comando =
presentación lista.

$ /frontend-slides Genera el informe mensual de cartera usando los archivos en /contexto

Claude Code lee los archivos de contexto, entiende los datos y genera una presentación HTML completa, animada y con formato consistente — lista para abrir en el navegador.

03 · Cómo funciona

La skill tiene 7 pasos automáticos.

01
Detecta el modo
Nueva presentación, conversión PPT, o mejora de slides existentes.
02
Lee el contexto
Analiza los archivos que le das: datos, notas, objetivos del informe.
03
Selecciona el estilo
12 presets visuales. Tú eliges el feeling, Claude aplica el diseño.
04
Construye y valida
Genera el HTML, verifica que todo encaje en pantalla, sin scroll interno.
04 · Archivos de contexto

Estructura tu carpeta
de contexto así.

📁 informe-cartera/
  📁 contexto/
    📄 resumen.md ← descripción del informe
    📊 datos.csv ← métricas clave
    📄 hallazgos.md ← insights del equipo
    📄 audiencia.md ← para quién es
  📁 assets/ ← imágenes opcionales
resumen.md
Propósito del informe, período, decisiones que se tomarán con él.
datos.csv / .xlsx
Cifras concretas. Claude las lee, las interpreta y las convierte en slides.
hallazgos.md
Lo que el equipo ya analizó. Claude lo presenta con jerarquía visual.
05 · El comando
Paso a paso

Así se invoca la skill
desde Claude Code.

# Opción 1: descripción libre
$ /frontend-slides Genera informe cartera usando contexto/

# Opción 2: especificar el preset visual
$ /frontend-slides Informe mensual, preset "Notebook Tabs", tone corporativo

# Opción 3: convertir PPT existente
$ /frontend-slides Convierte informe-julio.pptx y mejora el diseño

Claude Code lee los archivos, pregunta solo lo mínimo necesario y genera el HTML completo con navegación por teclado, animaciones y responsive.

06 · Resultado

Un archivo HTML.
Cero dependencias.
Todo adentro.

  • ▹
    Navegación completa — teclado (←→), rueda del mouse, swipe en móvil, barra de progreso.
  • ▹
    Animaciones en entrada — cada elemento aparece con timing calculado, sin ser ruidoso.
  • ▹
    100% responsive — se ve igual en 4K, laptop, tablet y celular.
  • ▹
    Reducción de movimiento — respeta prefers-reduced-motion automáticamente.
  • ▹
    Abre directo en el browser — doble clic al archivo. Sin instalar nada, sin servidor.

09 · Flujo de trabajo

El flujo completo
para tu equipo.

📁
1. Prepara
Centraliza datos, hallazgos y objetivos en archivos .md / .csv
→
⌨️
2. Invoca
/frontend-slides
+ descripción del informe
→
🎨
3. Elige estilo
Claude genera 3 previews visuales o aplica el preset del equipo
→
✅
4. Listo
HTML completo, abre en navegador, comparte o exporta

Tip para el equipo: guarda un preset fijo en las instrucciones del proyecto (CLAUDE.md) para que todos los informes salgan con el mismo look sin tener que pedirlo cada vez.
